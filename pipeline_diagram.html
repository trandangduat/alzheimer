Input Reorientation

Chuẩn hoá định hướng và không gian ảnh MRI
mri_convert
▼

📥 Input

OAS1_{sid}_MR1_mpr_n4_anon_sbj_111.img

📤 Output

mri/orig.mgz
mri/transforms/talairach.xfm

⚙️ Các lệnh thực hiện

CMDmri_convert --conform input.img mri/orig.mgz
Resample → 1mm isotropic, 256³, LIA orientation
CMDtalairach_avi --i orig.mgz --xfm talairach.xfm
Tạo Talairach transformation matrix
2
✨

Image Standardization

N4 Bias Correction + Intensity Normalization
SimpleITK + mri_normalize
▼

📥 Input

mri/orig.mgz

📤 Output

mri/nu.mgz
mri/T1.mgz

⚙️ Các lệnh thực hiện

LIBSimpleITK.N4BiasFieldCorrectionImageFilter()
Loại bỏ bias field, iterations=[20,20,20] → nu.mgz
CMDmri_normalize -g 1 -seed 1234 -mprage nu.mgz T1.mgz
Chuẩn hoá cường độ ảnh → T1.mgz

Cortical Surface Registration

Đăng ký bề mặt lên atlas chuẩn fsaverage
SUGAR (Docker)
▼

📥 Input

surf/lh.sphere
surf/lh.curv
surf/lh.sulc

📤 Output

surf/lh.sphere.reg

⚙️ Các lệnh thực hiện

CMDmris_inflate lh.white lh.inflated
Inflate white surface → inflated + sulc
CMDmris_sphere -w 0 lh.inflated lh.sphere
Project inflated surface lên sphere
DOCKERdocker run ninganme/sugar:latest --sid {sid} --hemi lh --device cuda
SUGAR deep-learning registration → lh.sphere.reg

Anatomical Stats

Tính toán chỉ số hình thái học từng vùng não
mris_anatomical_stats
▼

📥 Input

label/lh.aparc.annot
surf/lh.pial
surf/lh.white

📤 Output

stats/lh.aparc.stats
surf/lh.thickness

⚙️ Các lệnh thực hiện

CMDmris_thickness {sid} lh lh.thickness
Tính cortical thickness (khoảng cách white↔pial)
CMDmris_anatomical_stats -noglobal -a lh.aparc.annot -f lh.aparc.stats {sid} lh pial
→ ThickAvg, SurfArea, GrayVol, CurvInd cho 34 ROIs
